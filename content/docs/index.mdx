# Welcome to the CPACC Study Guide!

This guide is designed to help you prepare for the **Certified Professional in Accessibility Core Competencies (CPACC)** exam. You'll find structured study materials, key concepts, and resources to ensure you're well-prepared.

## How to use this guide

- Explore different sections to learn about key accessibility principles.
- Review key terms, laws, and guidelines related to accessibility.
- Practice with quizzes and review materials.
